# Fake Email Detector

A machine learning project to detect spam emails using NLP and multiple classifiers.

## Features
- Preprocess email text with tokenization, stopword removal, and stemming
- Feature extraction using TF-IDF
- Compare multiple models: Naive Bayes, Logistic Regression, SVM
- Best model selection
- Probability-based confidence scoring for predictions
- Model saving/loading with joblib
- Custom email testing

## Requirements
- Python 3.x
- pandas
- numpy
- nltk
- scikit-learn
- joblib
